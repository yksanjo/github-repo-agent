# Fork Prompts Documentation

This directory contains detailed prompts and guidance for forking and adapting 10 high-value open-source repositories to align with our existing projects: Music AI Lab, ai-finance-coach-mvp, github-repo-automation, and chatbot repositories.

## Overview

Each fork prompt document provides:
- Repository overview and current status
- Use case alignment with existing projects
- Forking strategy and customization roadmap
- Contributor guidelines and technical requirements

## Repository Fork Prompts

### Music AI & Audio Processing

1. **[YuE Music Generation](./yue-music-generation-fork-prompt.md)** - Open-source full-song music generation model
   - Generates complete songs with vocals and accompaniment from lyrics
   - Apache 2.0 licensed
   - Perfect for Music AI Lab integration

2. **[Microsoft Muzic](./microsoft-muzic-fork-prompt.md)** - Comprehensive AI music research project
   - Includes MusicBERT, Museformer, and other SOTA models
   - Well-documented with multiple sub-projects
   - Great for extending music understanding and generation features

3. **[MuseGAN](./musegan-fork-prompt.md)** - Multi-track music generation with GANs
   - Generates polyphonic music for multiple instruments
   - Adaptable for music agreement analysis workflows

### AI Chatbots & NLP

4. **[Rasa](./rasa-chatbot-fork-prompt.md)** - Open-source conversational AI framework
   - Enterprise-grade chatbot platform (20.9k stars)
   - Excellent for building domain-specific chatbots
   - Specialized bots for music contract analysis or finance coaching

5. **[Phidata](./phidata-agents-fork-prompt.md)** - Multi-modal AI agents with memory and tools
   - Build agents that work together to solve problems
   - Beautiful Agent UI included
   - Perfect for finance coaching or music industry assistants

### Automation & Developer Tools

6. **[Skyvern](./skyvern-automation-fork-prompt.md)** - Browser automation with LLMs and computer vision
   - Automates workflows without brittle XPath-based scripts
   - Great for automating music industry tasks
   - Could enhance github-repo-automation project

7. **[OpenHands](./openhands-coding-fork-prompt.md)** - AI coding assistant (52.6k stars)
   - Creates and edits files, executes commands
   - Could be integrated with automation tools
   - Useful for automating repetitive coding tasks

8. **[Cline](./cline-ide-agent-fork-prompt.md)** - Autonomous AI coding agent for IDEs
   - Uses Claude 3.5 Sonnet for agentic coding
   - Model Context Protocol (MCP) support
   - Perfect for extending development workflow

### Finance & Data Analysis

9. **[Streamlit](./streamlit-finance-fork-prompt.md)** - Build data apps quickly (38.7k stars)
   - Perfect for creating interactive finance coaching dashboards
   - Easy to prototype and share AI-powered financial tools
   - Could transform finance coach MVP into full-featured app

10. **[Appsmith](./appsmith-tools-fork-prompt.md)** - Low-code platform for internal tools (36.6k stars)
    - Build admin panels and dashboards quickly
    - Great for creating music industry management tools
    - Could be used to build internal tools for Music AI Lab
